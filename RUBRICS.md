## 📊 Marks Calculation Summary

### Components with Caps:

- **Community Service**: Maximum 40 marks
- **Workshops & Seminars**: Maximum 20 marks
- **Online Courses**: Maximum 80 marks
- **Coding Platforms**: Maximum 120 marks
- **Minor Projects**: Maximum 160 marks

### Components with Full FA Marks (240):

Only the **highest scoring** from these applies:

- Patent Filing with Prototype + IPM Publication: 240
- Technology Transfer/Commercialization: 240
- Scopus Paper Publication: 240
- Winning Industry/Government Hackathon: 240
- Entrepreneurship (Udyam/DPIIT/Funding): 240

### Formula:

```
Total Marks = Community Service (max 40)
            + Full FA Component (max 240 - highest among eligible)
            + Workshops (max 20)
            + Online Courses (max 80)
            + Coding Platforms (max 120)
            + Minor Projects (max 160)
            + Other Hackathons/Competitions
```

**Maximum Possible Total**: Varies based on combination of activities

## 🔍 Proof Document Requirements

All entries must include supporting proof documents:

- Certificates
- Letters of Appreciation
- Publication Links
- Patent Application Numbers
- Registration Certificates
- Screenshots (for coding platforms)
- Project Links (GitHub, Demo)

## ✅ Validation Rules

1. **Team Size**: Maximum 3 members for community service
2. **NIRF Rank**: Must be within specified limits (Top 200/300)
3. **Semester-wise**: Coding platform scores apply only to current semester
4. **Full FA**: Only one Full FA component counts (highest value)
5. **Proof**: All entries require valid proof documents
