# Internal Marks Calculator - User Guide

## 🎯 Quick Start

This system calculates internal marks based on 9 different activity components. Follow this guide to add entries and track marks.

---

## 📝 How to Add Entries

### Step 1: Add Student

1. Go to **Students** page
2. Click **Add New Student**
3. Fill in details:
   - Student Name
   - Roll Number (Unique)
   - Semester (2-7)
   - Email
   - Department
4. Click **Add Student**

### Step 2: Add Activities

1. Click on student name to view details
2. Select the activity tab
3. Click **Add Entry**
4. Fill in ALL required details
5. Upload proof document
6. Submit

### Step 3: View Marks

- Marks are calculated automatically
- View breakdown in student details page
- Check total marks in Reports section

## 📋 Component Details & What to Enter

### 1️⃣ Community Service (40 marks max)

**When to use**: Conducting workshops, coding sessions, or training at schools/NGOs/industries

**Required Details**:

- ✅ Organization Name (Full name of school/NGO/industry)
- ✅ Activity Type (Workshop/Coding Session/Training Program)
- ✅ Detailed Description (What was taught, topic covered, impact)
- ✅ Team Size (1 to 3 members only)
- ✅ Date Conducted
- ✅ Duration (Hours/Days)
- ✅ Number of Participants benefited
- ✅ Proof: Certificate/Letter from organization

**Marks**: 40 marks if team size ≤ 3

**Example Entry**:

```
Organization: ABC Public School, Chennai
Activity: Python Programming Workshop
Description: Conducted 3-day workshop on Python basics for Class 11 students.
              Covered variables, loops, functions. 50 students participated.
Team Size: 3
Date: 15-Feb-2026
Duration: 3 days (6 hours total)
Proof: Certificate_ABC_School.pdf
```

### 2️⃣ Patent Filing & Prototyping (240 marks - Full FA)

**When to use**: Filed patent with working prototype through institution's IPM cell

**Required Details**:

- ✅ Patent Title (Exact as in application)
- ✅ Application Number (From IPM cell)
- ✅ Filing Date
- ✅ Status (Filed/Published/Granted)

- ✅ Commercialized? (Yes/No)

- ✅ Detailed Description of invention
- ✅ Problem it solves
- ✅ Proof: Patent application/IPM approval letter

**Marks**:

- 240 if: Prototype + Patent Filed + IPM approval
- 240 if: Technology Transfer OR Commercialization

**Example Entry**:

```
Title: IoT-Based Smart Irrigation System
Application Number: 202341012345
Filing Date: 10-Jan-2026
Status: Filed
Prototype: Yes (Working model with sensors and mobile app)
Technology Transfer: No
Commercialization: No

Problem Solved: Reduces water wastage in agriculture by 40%
Proof: IPM_Approval_Letter.pdf, Patent_Application.pdf
```

## 🎯 Tips for Maximum Marks

1. **Plan Activities**: Target Full FA components (Patents, Scopus, Entrepreneurship)
2. **Document Everything**: Keep all certificates, screenshots, emails
3. **Team Activities**: Form teams of max 3 for community service
4. **Regular Updates**: Update coding platform scores every semester
5. **Quality Over Quantity**: One Scopus paper (240) > Multiple workshops (20 each)
6. **Proof is Mandatory**: No proof = No marks
7. **Semester Matters**: For coding platforms, marks vary by semester
8. **NIRF Verification**: Verify NIRF ranks from official website before entering

## 📊 Understanding Marks Calculation

### Capped Components:

Your total in these categories cannot exceed the cap:

- Community Service: Max 40
- Workshops: Max 20
- Online Courses: Max 80
- Coding Platforms: Max 120
- Minor Projects: Max 160

### Full FA Components (240 marks):

Only **ONE** of these counts (the highest):

- Patent with Prototype
- Scopus Paper
- Winning Government/Industry Hackathon
- Entrepreneurship (Udyam/DPIIT/Funding)

### Example Calculation:

```
Community Service: 40 (achieved 40, capped at 40)
Patent: 240 (achieved 240) ← Full FA
Workshops: 20 (achieved 20, capped at 20)
Online Courses: 60 (achieved 100, capped at 80)
Coding Platforms: 80 (achieved 80, capped at 120)
Hackathons: 20 (achieved 20, but patent already gives Full FA)
Minor Projects: 20 (achieved 20, capped at 160)

Total = 40 + 240 + 20 + 60 + 80 + 20 = 460 marks
(Hackathon 20 added as it doesn't give Full FA)
```

## 🆘 Common Issues

**Q: I added an entry but got 0 marks**
A: Check if you meet ALL criteria (team size, NIRF rank, proof document, etc.)

**Q: My coding score isn't giving marks**
A: Verify your semester matches the scoring rubrics. Sem 2 students won't get marks on CodeChef.

**Q: I won a hackathon but only got 160 marks, not 240**
A: 240 is only for Industry/Government organized hackathons. Regular hackathon wins = 160.

**Q: Can I get marks for both Patent and Scopus?**
A: Only the highest Full FA component counts. System picks max(240, 240) = 240 once.

**Q: My workshop at NIRF rank 250 institution gave 0 marks**
A: Workshops require Top 200 NIRF. Rank 250 doesn't qualify.
